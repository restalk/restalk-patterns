Contribute

We would be happy if you want to contribute to this pattern language, may it be you want to correct a typo, add known uses, make pattern suggestions or anything else. In that case you have the following options:

  • Open or comment on any of the issues on GitHub.
  • Fork the repository and create a pull request.
  • Write an e-mail to mail@restalk-patterns.org.
  • Follow us on Twitter: @RESTalkPatterns.
  • Create your own patterns or conversation models by using this SVG file as starting point.
